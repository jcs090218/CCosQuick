CQ_Animation

Play animation frame by frame with the same interval in SPF.

Variables

NameDescription
spriteSprite renderer that will display our sprite frame.
loopLoop for this animation.
SPFSeconds per frame.
framesList of sprite that will be display as a frame.

Functions

NameDescription
playPlay the animation from the START frame.
stopStop the animation.
pausePause the animation.
unPauseUnpause the animation.
